Cutesy checks HTML documents for consistency and best practices. It includes a set of rules, most of which can be fixed automatically.


## Testing, etc.

Install development requirements (Requires Python >= 3.8):

    make install

Sort imports:

    make format

Lint:

    make lint

Test:

    make test
